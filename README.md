# coding_badly
Figuring out how to do anything at all.

This is a centralized storage space for all my projects while I learn to code

## Roadmap for Tourists:
There's a lot of stuff in here, and organizing all of it is a more monumental task than I currently have time for, so here's a list of some cool stuff to look at:
1. The `/logic_challenges` folder contains:
    - my AP Computer Science Principles create project
    - a custom-built rot cipher decoder/encoder
    - some Katiss programming challenges in `/2021` and `/2022`
    - a click-per-second game which for some reason I called `colors`
2. The `/MathStuff` folder contains:
    - a number of (semi-functional) custom-built circle-based little physics engines in `/again` and `/Particle_Collision`
    - a wildly inneficient but quite fun flow field renderer and creator in `/flow_fields`
    - an attempt at a decently optimized factor finder in `/minis`
3. The `/html_coding` folder contains:
    - a number of fairly uninteresting projects which did teach me a bit about web dev
    - the beginnings of a remake of Universal Paperclips (which you should look up, its a very fun incremental clicker game) in `/my_paperclips`
4. The `/clones` folder contains:
    - a couple iterations of n-mino Tetris clones in the `/PolyminoTests` folders
    - a Snake clone, a generic Tower Defense game implementation, and the beginnings of an open-world 2d voxel game (in `/walkin`)
    - the beginnings of a game launcher in `hub.py` to be able to access and run those all from the same place
5. Similar, but generally smaller scale projects in other languages:
    - Rust in `/rusting` (`/drawing_practice` is probably the most interesting)
    - Java in `/JavaTesting`
    - react in `/react` (I have some better projects in react, which were made elsewhere)
6. `/p5_trials`, `/raylib` include some attempts at using other rendering methods to do some low-level stuff and interact with graphics shaders

**Note: there is some content in this repository which has been directly copied in from outside sources as reference, and is not my own work. I've tried to mark each of these clearly with the location they were sourced from.
