# Game Design Document: Universal Continuity

Universal paperclips is a game about doing one simple thing, and never ever stopping

This has a very similar premise:
Time is happening, make it stop

In particular, you are experiencing the passage of time compared to everything else you can observe, you want that to not be happening
Simplest way I know to do this, go fast.


## Functionality:
It is a progressive idle/clicker game which contains multiple stages, internal minigames, meta commentary, and some slightly narrative elements
